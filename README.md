### Running locally

- Clone the repository
 ```
 git clone https://github.com/listikrazmarinoviy/project_canopus.git
 ```

- Enter the directory project
  ```
  cd project_canopus
  ```

- Install all dependencies
  ```
  npm install
  ```

- Start dev server
  ```
  npm run dev
  ```
